

# Обработчик задач на эллиптических кривых

Эта программа выполняет вычисления на эллиптических кривых по данным из текстовых файлов.  
Все `.txt` файлы из каталога `INPUT/` автоматически считываются, обрабатываются,  
а результаты записываются в каталог `OUTPUT/` с тем же именем файла и добавлением суффикса `_OUTPUT`.

Пример:
- `INPUT/test.txt` → `OUTPUT/test_OUTPUT.txt`

---

## Формат входных данных

Пустые строки игнорируются.  
Числа можно записывать в любом формате:
- десятичном (`123`)
- двоичном (`0b1010`)
- восьмеричном (`0o77`)
- шестнадцатеричном (`0x1a2b`)

---

### Кривые над простыми полями (Z_p)

```

Z_p

<p>
<a> <b>   # можно писать на одной строке или на отдельных строках
<задачи...>
```

Также допускаются обозначения: `1`, `Zp`, `zp`, `Fp`, `prime`.

Уравнение имеет вид:
`y^2 = x^3 + a*x + b (mod p)`

---

### Кривые над бинарными полями GF(2^n)

```
GF(2^n)
<список степеней многочлена>
<a> <b>
<задачи...>
```

Также допускаются варианты: `2`, `3`, `GF2S`, `GF2NS`, `GF(2^...)`.

Пример:

```
GF(2^n)
163 7 6 3 0
0x1
0x1
M (0x2fe13c..., 0x28907...) 2
```

Строка со степенями задаёт редукционный (неприводимый) многочлен.
Например, `163 7 6 3 0` означает многочлен
`x^163 + x^7 + x^6 + x^3 + 1`.

Уравнение для бинарных кривых:
`y^2 + x*y = x^3 + a*x^2 + b`

---

### Поддерживаемые задачи

В каждой строке описана одна операция:

* Сложение точек

  ```
  A (x1, y1) (x2, y2)
  ```

  (также допускается формат с запятой: `A (x1,y1), (x2,y2)`)

* Умножение точки на число

  ```
  M (x, y) k
  ```

---

### Формат вывода

Результат для каждой задачи записывается в виде:

```
(3, 1) + (5, 1) = (9, 16)
(3, 1) * 5 = (5, 1)
```

Если результатом является **точка на бесконечности**, она обозначается как `O`.

---

## Запуск программы

Из папки, где лежит файл `main.py`, выполнить команду:

**Windows**

```bash
python main.py
```

Появится интуивтивно понятный графический интерфейс.

---

## Результаты работы

Все входные файлы: `INPUT/*.txt`
Все результаты: `OUTPUT/*_OUTPUT.txt`

---

## Пример

Файл `INPUT/example.txt`:

```
1
17
2 2
A (3,1) (5,1)
M (3,1) 5
```

После запуска появится файл `OUTPUT/example_OUTPUT.txt`:

```
(3, 1) + (5, 1) = (9, 16)
(3, 1) * 5 = (5, 1)
```

---

## Возможные ошибки

* Файл не обработался — проверьте, что у него расширение `.txt`.
* Мало строк в файле — проверь, что указаны тип кривой, параметры и хотя бы одна задача.
